# Estado actual
- el proyecto tiene sus textos en español

# Lo que se busca
a partir de esos textos que encontraste, créame un archivo en src/messages/es.json bajo la siguiente estructura

{
"Index": {
"title": "Hola mundo",
"description": "Bienvenido a mi aplicación"
}
},
....

### donde:
- Index: es el texto que encontraste, pero en minúsculas, los espacios deben ser reemplazados con _ , las vocales con acento deben ser reemplazadas por vocales sin acento, y todo lo adicional que no sea a-z debe ser eliminado
- title: es el texto que encontraste
- descripción: es una descripción de lo que quiere decir el texto
